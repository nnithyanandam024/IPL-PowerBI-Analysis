# 🏏 IPL Data Analysis using Power BI

---

##  Project Overview
This project focuses on analyzing **Indian Premier League (IPL)** match data using **Microsoft Power BI**.  
The main objective of the project is to transform raw IPL match data into **meaningful visual insights** through an interactive dashboard.

The dashboard helps in understanding:

- Team performance
- Player achievements
- Match distribution across seasons and venues

This project demonstrates how **data visualization tools** can be used to simplify complex datasets and generate valuable insights.

---

##  Objectives

- Analyze IPL match data using **data visualization techniques**
- Identify the **teams with the highest number of wins**
- Examine **season-wise match distribution**
- Identify players who won the most **Player of the Match awards**
- Understand the **distribution of matches across different stadiums**

---

##  Tools and Technologies

- **Microsoft Power BI**
- **CSV Dataset**
- **Data Visualization Techniques**

---

##  Dashboard Features

The Power BI dashboard includes the following visualizations:

### 1️) Total Matches Played
Displays the **total number of IPL matches** in the dataset.

### 2️) Team-wise Match Wins
Shows the **number of matches won by each team**.

### 3️) Matches Played per Season
Displays **match distribution across different IPL seasons**.

### 4️) Top 10 Player of the Match Winners
Highlights the players who received the most **Player of the Match awards**.

### 5️) Matches by Stadium
Shows how matches are distributed across **different venues**.

### 6️) Season Filter
Allows users to **filter the dashboard by IPL season**.

---

##  Key Insights

- Certain teams consistently perform better across multiple seasons.
- Some players dominate the **Player of the Match awards** due to consistent performances.
- Matches are distributed across many stadiums, showing IPL's **nationwide reach**.
- Seasonal analysis helps understand how the **tournament has evolved over time**.

##  Dataset Information

### Dataset
**IPL Complete Dataset (2008–2024)**

This dataset contains detailed information about all matches played in the **Indian Premier League (IPL)** from **2008 to 2024**.

The data is useful for analyzing:

- Team performance
- Player statistics
- Match trends across different IPL seasons

---

### Data Source
The dataset is sourced from **Cricsheet** and made available through the **Kaggle platform**.

---

### Dataset Files

The dataset consists of **two main CSV files**:

#### 1️) matches.csv
This file contains **match-level information** such as:

- Match ID
- Season
- Teams playing the match
- Toss winner
- Match winner
- Player of the Match
- Venue
- City
- Match result

This file is mainly used for **overall match analysis**.

---

#### 2️) deliveries.csv
This file contains **ball-by-ball details** of each match.

Important columns include:

- Match ID
- Inning number
- Batting team
- Bowling team
- Over number
- Ball number
- Batter
- Bowler
- Runs scored by batter

This dataset helps analyze **detailed match events and player performance**.

---

### Dataset Size

- **Deliveries dataset:** ~27 MB  
- Contains more than **300,000 ball-by-ball records**

---

### Dataset Usage

This dataset is used to analyze:

- Team performance across seasons
- Player of the Match statistics
- Match distribution by stadium
- Seasonal match trends
